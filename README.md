# ghost-namer

An app that provides the user a ghost name.

Routing and logic in `main.py`. Datastore helper functions in `lib\helper.py`.

## Stack
- Google App Engine
- Cloud Datastore
- Firebase Auth
- Flask
- Flask_Session + Redis
